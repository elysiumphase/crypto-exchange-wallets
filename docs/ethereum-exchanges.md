## Key naming rule

Exchanges' key as defined in `walletsByExchange.js` is formatted in kebab lower case except for `.` character which is kept as it is. For an example `Binance US` will become `binance-us` and `Gate.io` will be `gate.io` and NOT `gate-io`.
